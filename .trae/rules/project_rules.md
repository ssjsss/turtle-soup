# 海龟汤题库分类规则

## 搞笑
- 不能有死人
- 答案很无厘头，出人意料且荒诞

## 脑洞
- 有正常生活里没有的元素
- 例如：幽灵、超能力、未来科技、时间穿越、人格实体化等超现实设定

## 恐怖
- 有死人

## 优先级
脑洞 > 恐怖 > 搞笑

即：含超现实元素的优先归脑洞；有超现实+有死人也归脑洞；有死人无超现实归恐怖；无死人无超现实且荒诞归搞笑。

# Git 提交规范

- 类型：feat/fix/docs/style/refactor/perf/test/build/ci/chore
- 格式：`<type>(<scope>): <subject>`，subject 中文、不超过 50 字、结尾不加标点
- 示例：`feat(user): 新增用户登录接口`
